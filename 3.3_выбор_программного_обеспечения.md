# 3.3 Выбор программного обеспечения

Выбор технологий для разработки веб-приложения осуществлялся с учетом требований к функциональности, производительности, масштабируемости и удобству разработки.

## 3.3.1 Frontend Framework: Next.js 14

**Обоснование выбора:**

Next.js 14 выбран как основной фреймворк для разработки по следующим причинам:

1. **Server-Side Rendering (SSR)** — обеспечивает быструю загрузку страниц и улучшает SEO
2. **App Router** — современная система маршрутизации с поддержкой серверных компонентов
3. **API Routes** — встроенная поддержка создания API endpoints без отдельного backend
4. **Оптимизация изображений** — встроенный компонент Image для автоматической оптимизации
5. **TypeScript поддержка** — нативная поддержка TypeScript для типобезопасности
6. **Большое сообщество** — обширная документация и библиотеки

## 3.3.2 Язык программирования: TypeScript

**Обоснование выбора:**

TypeScript выбран для повышения надежности кода:

1. **Статическая типизация** — выявление ошибок на этапе компиляции
2. **Улучшенная поддержка IDE** — автодополнение и рефакторинг
3. **Лучшая читаемость кода** — явные типы делают код более понятным
4. **Совместимость с JavaScript** — возможность использования существующих библиотек

## 3.3.3 База данных: SQLite с Prisma ORM

**Обоснование выбора SQLite:**

1. **Легковесность** — не требует отдельного сервера базы данных
2. **Простота развертывания** — база данных хранится в одном файле
3. **Достаточная производительность** — для средних объемов данных
4. **ACID транзакции** — обеспечение целостности данных
5. **Бесплатность** — отсутствие лицензионных ограничений

**Обоснование выбора Prisma:**

1. **Type-safe запросы** — автоматическая генерация типов из схемы
2. **Миграции** — управление версиями схемы базы данных
3. **Удобный API** — интуитивный синтаксис для работы с данными
4. **Автодополнение** — поддержка в IDE
5. **Документация** — подробная документация и примеры

## 3.3.4 Стилизация: Tailwind CSS

**Обоснование выбора:**

1. **Utility-first подход** — быстрое создание интерфейсов
2. **Адаптивность** — встроенная поддержка responsive design
3. **Кастомизация** — легко настраиваемая тема
4. **Производительность** — автоматическое удаление неиспользуемых стилей
5. **Современный дизайн** — готовые компоненты и утилиты

## 3.3.5 3D графика: Three.js и React Three Fiber

**Обоснование выбора:**

1. **Three.js** — наиболее популярная библиотека для 3D графики в веб
2. **React Three Fiber** — React-обертка для Three.js, обеспечивающая декларативный подход
3. **Производительность** — оптимизированная работа с WebGL
4. **Поддержка форматов** — загрузка GLB/GLTF моделей
5. **Интерактивность** — поддержка контроллеров камеры и взаимодействия
